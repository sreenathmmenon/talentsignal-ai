# Project Tagline Decision

Date: June 16, 2026

## Final Tagline

Evidence-backed hiring decisions for any role.

## Why This Is The Right Project-Level Tagline

TalentSignal AI is not only a Redrob hackathon ranker and not only a resume-table UI. The product direction is a universal JD-to-hiring-decision engine.

This tagline focuses on one stable promise:

- Evidence-backed: the system ranks from inspectable candidate facts, not keyword matching.
- Hiring decisions: the product supports ranking, comparison, trust review, interview probes, and exports.
- Any role: the architecture is universal across JDs and categories, with the Redrob challenge JD as the first proof case.

## Rule

Do not change this tagline casually during implementation.

Only change it after an explicit product-positioning discussion and user approval.

UI labels, section names, and demo copy may evolve, but the project tagline should remain stable unless Sreenath explicitly asks to revisit it.
