# Final Submission Checklist

## Premortem Review Gate

Review `FIRST_PRIZE_PREMORTEM.md` before final submission.

Minimum checks:

- No obvious keyword stuffer in top 10.
- No non-tech AI-skill-only profile in top 10.
- No top-10 candidate lacks production ML/search/retrieval/ranking evidence.
- No stale/low-response profile outranks comparable active candidates without exceptional evidence.
- No hallucinated reasoning.
- No unsupported employer/skill claims in reasoning.
- No duplicate candidate IDs.
- No duplicate ranks.
- Scores are non-increasing.
- Official validator passes.
- Final command runs offline without network/GPU/API calls.
- Runtime is below 5 minutes.
- README reproduction command matches actual command.
